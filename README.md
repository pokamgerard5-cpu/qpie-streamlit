# QPIE - Probability Intelligence Engine

Real-time RNG analysis and prediction system for 2-player games.

## Features
- Click-based manual outcome entry
- Real-time probability predictions
- Automated PnL tracking
- Confidence scoring with skip recommendations
- Monte Carlo simulations

## Usage
1. Click "Manual Input" in sidebar
2. Click P1/P2/TIE buttons after each round
3. Watch predictions improve over time
4. Track PnL in separate dashboard

## Data Storage
All outcomes saved to `data/outcomes.csv`

## Model
Simple Bayesian model (upgradable to LSTM/XGBoost)
